# Sales-Performance-Analysis-Dashboard

Project Overview-
Provides a comprehensive view of sales performance for a computer hardware company operating in a changing market. It identifies key performance indicators, visualizes sales trends, and highlights actionable insights to help management improve decision-making and optimize business operations.

 Objectives-
- Analyze sales data to understand trends, regional performance, and customer behavior.  
- Identify high-performing and low-performing segments to guide resource allocation.  
- Evaluate efficiency and supply chain flow through visual analytics
- Support customer development and business growth with real-time insights.  

Key Features-
-  Interactive Power BI Dashboard: Dynamic visualizations including KPIs, maps, and trend charts.  
-  Data Cleaning + Preparation: Raw data was transformed and standardized using  SQL Workbench  and  Excel  before importing into Power BI.  
- Data Analysis: Statistical summaries and relationships between product categories, revenue, and customers.  
-  Insights Generation: Highlighted regions, products, and sales representatives driving the most revenue.  

Tools Used-
- Power BI for dashboard creation and visualization
- MySQL Workbench for data extraction + cleaning
- Excel for data processing
